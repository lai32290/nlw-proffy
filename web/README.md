# Proffy Frontend

## How to Run

Run the following command in the project folder terminal
```
# Moving into the web folder
$ code web

# Install the dependencies
$ npm install

# Run the server
$ npm start
```
Now the website is available in the [http://localhost:3000](http://localhost:3000)
