Proffy is a platform to connect teacher and studants, a place where people can choose what subject they want to teach/learn and what is the best day of the week they want to do it.

## Technologies Involved

### Backend
- Node.js
- Express
- SQLite
- Knex
- TypeScript

### Frontend
- React
- TypeScript
- React Router DOM
- Axios

### Mobile
- React Native
- Axios
